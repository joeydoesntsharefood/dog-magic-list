# US01.5: UI - Consulta Rápida de Preços (Frontend)

**Descrição:** Implementar a interface de usuário para consulta direta de preços, integrada ao backend REST.

## Componentes a Criar:
1.  **SearchBox:** Input de texto com botão de busca estilizado com a paleta `tattoo-red`.
2.  **PriceCard:** Card para exibir a imagem da carta e os valores (USD e BRL).
3.  **QuickViewContainer:** Layout de navegação para alternar entre "Minhas Listas" e "Consulta Rápida".

## Comportamento da UI:
1.  **Busca:** Ao pesquisar, exibir um skeleton loader ou spinner enquanto a chamada `GET /prices` é processada.
2.  **Resultados:** Exibir o valor em BRL com destaque (ex: R$ 25,00) utilizando a cor `aged-bronze` para o símbolo da moeda.
3.  **Link Externo:** Botão para abrir o link da LigaMagic no navegador externo.
4.  **Erro:** Tratar 404 exibindo "Carta não encontrada no Multiverso".

## Tarefas de Implementação:
-   [ ] Criar estado `activeTab` no `App.tsx` para alternar entre Listas e Consulta.
-   [ ] Criar a função `fetchPrice(cardName)` no front.
-   [ ] Implementar o layout responsivo do card de resultado.
-   [ ] Escrever testes Vitest simulando a busca de preço e renderização do card.

## Metodologia TDI:
-   **Teste de UI:** Garantir que ao digitar e clicar em "Pesquisar", o `fetch` é chamado com o nome correto.
-   **Teste de Visualização:** Validar que o valor convertido em BRL aparece na tela após a resolução da promise.
