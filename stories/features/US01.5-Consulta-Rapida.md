# US01.5: Consulta Rápida de Preços (Market Quick-Check)

**Descrição:** Como usuário, desejo pesquisar o valor de uma carta de forma direta e rápida para ter uma estimativa de preço em Reais (BRL), utilizando a conversão automática do valor de mercado internacional.

## Critérios de Aceite (AC):
1.  O sistema deve possuir uma aba/seção de "Consulta Rápida".
2.  Ao digitar o nome da carta, o sistema deve buscar os dados no Scryfall.
3.  O sistema deve buscar a cotação atual do USD/BRL via AwesomeAPI.
4.  O preço final exibido deve ser: `Preço Scryfall (USD) * Cotação AwesomeAPI (BRL)`.
5.  A interface deve exibir a imagem da carta, o preço em BRL em destaque e um botão de "Ver na LigaMagic" (Link externo).
6.  Todo o processo deve ser gratuito e não exigir chaves de API pagas.

## Estratégia TDI:
*   **Teste Unitário (Backend - PriceService):** Validar o cálculo de conversão (ex: $1.00 * R$ 5.00 = R$ 5,00).
*   **Teste Unitário (Integration Parser):** Validar se o sistema trata corretamente cartas sem preço (ex: cartas promocionais ou muito raras).
*   **Teste de Integração:** Validar o fluxo completo desde a digitação até a exibição do valor convertido na UI.
